# 执行计划：添加左侧阅读设置侧边栏

## 目标
1. 在阅读视图左侧添加一个设置面板，包含“主题”和“宽度”的切换按钮。
2. 实现实时切换主题（支持浅色/深色/护眼等）。
3. 实现实时切换宽度（默认、较宽、全屏）。

## 具体方案
1. **修改 `web/index.html`**：
   - 在 `body` 中增加一个左侧固定的设置面板 `<div class="sidebar">...</div>`。
   - 包含主题选择（浅色、深色、护眼）。
   - 包含宽度选择（默认、较宽、全屏）。

2. **修改 `web/styles.css`**：
   - 提取并增加不同主题的 CSS 变量。默认是 `dark`，增加 `light`（浅色）、`sepia`（护眼）的主题样式。
   - 增加宽度的类控制（如 `.width-default` 720px, `.width-wide` 1000px, `.width-full` 100%）。
   - 实现侧边栏的样式 `.sidebar`，固定在左侧。

3. **修改 `web/app.js`**：
   - 添加改变主题和宽度的函数。
   - 将用户选择绑定到 `localStorage` 中以便刷新记忆。
   - 实现点击按钮后切换对应的类名或内联样式。
